# fsoft2026_1DA_3

Projeto desenvolvido no âmbito da unidade curricular de Fundamentos de Desenvolvimento de Software — LETI, ISEP 2025/2026.

## Tema
Loja de Música

## Descrição
Aplicação em C++ para gestão de uma loja de música, focada na venda de música em diversos formatos de mídia (CD, Vinyl, Cassete, Digital), gestão de artistas, álbuns, clientes e ratings.
